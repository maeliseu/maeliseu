Projeto 1

Sistema de Apparking

  • Category: API - Microserviço

Considerações

Durante o desenvolvimento do projeto tivemos a ideia de abordar a tecnologia onde exercesse o nó de comunicação e armazenamento das informações que seria alimentada e consumida por diversas tecnologias que é a Application Programming Interface (API) possibilitando benefícios como a segurança dos dados entre outras, desenvolvida utilizando a linguagem de programação Python com o framework Django e Django Rest. Por meio desse projeto pudemos melhor compreender a implementação com este conjunto de tecnologias, que atualmente são de extrema relevância para a manipulação segura de informações em diversas aplicações.
