Projeto 2

Combate-COVID19

  • Categoria: PHP - Laravel

Considerações

O projeto destinou-se a exercitar os conhecimentos aprendidos do Framework Laravel nas aulas do Curso Sistemas para Internet. Por isso foi idealizado um aplicativo WEB simples, para cadastrar e mostrar a totalização dos casos positivos de COVID19, sem necessidade de controles e de usuários.

A tarefa foi muito prazerosa, o framework disponibiliza ferramentas poderosas para auxiliar no trato com o banco de dados, na geração de formulários para alimentar os dados e na visualização dos dados tratados, neste caso fazendo as totalizações por regiões e inserindo em uma imagem da cidade separada por regiões os valores.

Mesmo não sendo um projeto de tamanho e complexidade alta, ficou claro o poder da programação em padrões, com a programação no padrão Model View Controller que separa as competências obtém-se um código enxuto de fácil entendimento e facilidade na manutenção, mesmo em códigos complexos e por pessoas que não participaram da criação do código.
